# 🎨 K-Food-Dictionary-server

이 웹페이지는 외국인들에게 한국의 다양하고 맛있는 음식을 소개합니다.

1. 사용자가 한국음식을 선택해서 정보를 받을 수 있다.

2. 사용자가 원하는 음식을 선택해 Mypage에 보관할 수 있다.

3. 사용자가 원하는 한국음식을 sns에 공유할 수 있다.
